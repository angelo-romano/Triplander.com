About Triplander.com

Triplander.com is an open source project created by Angelo Romano, an Italian Web/software developer. The project was originally designed in 2007 with the aim to build a innovative, intelligent travel-agency Web system capable to help customers in their search for (both actual and potential) favourite destinations. The project was successively started being implemented in the summer of 2008, with the first public beta being published later in September of the same year.
The website currently features only European cities and countries (including also Iceland, Greenland, Israel, Armenia, Azerbaijan and Georgia, plus a very few Asiatic Russia cities). It is, of course, a merely temporary choice made in order to deal with a reasonable workload; we want to offer a service that works, and it is obviously much easier to test it with a limited number of dynamic contents. We aim to extend the service at least to North American cities in the near future, and the whole world once the system is out of beta.
Triplander.com is still under quite a heavy development, and the currently published system is just a short overview of all the features we aim to offer in the future. As you can probably understand, this website is a dramatic work-in-progress; so, stay tuned and, why not?, in the meantime look for your town at Triplander.com!

Technical aspects

Triplander.com is based on a number of innovative open source applications, whose choice is definitely not a random one. The main system was built with Django, a Python-based framework for well-sized applications which offers the opportunity to code and develop new features in a very extensive and nonetheless simple way.
Triplander.com also uses jQuery to deal with all the JavaScript & AJAX clutter around, and PostgreSQL as DBMS – with PostGIS extensions, of course.
